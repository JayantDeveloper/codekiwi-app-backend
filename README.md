# CodeKiwi Backend (Slide-Syncer)

> The core server behind [CodeKiwi](https://codekiwi.tech), enabling real-time code and slide synchronization between teachers and students.

## Overview
This backend manages:
- WebSocket connections for real-time classroom updates  
- PDF upload and conversion for slides  
- Session management and data endpoints for the frontend  
- Optional containerized code execution for live teaching demos  

Built with **Express.js**, **WebSocket**, and **Node.js**, the backend connects all components of the CodeKiwi classroom experience.

## Role in the CodeKiwi Ecosystem
The backend powers both the classroom frontend and the teacher-facing web portal.
